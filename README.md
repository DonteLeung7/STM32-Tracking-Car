\# STM32 Intelligent Tracking Car



\## 项目简介



这是一个基于**STM**32f103c8t6 的智能循迹小车。



\### 功能



\- 五路红外循迹

\- 蓝牙遥控

\- 超声波避障

\- PWM电机驱动

\- 电机速度控制



\---



\## 开发环境



Keil5(ARM  Complier v5.06)



STM32 Standard Peripheral Library



\---



\## 硬件



STM32F103C8T6



HC05蓝牙



TB6612驱动



五路红外循迹模块（TCRT5000红外反射传感器）



超声波模块



\---



\## 项目结构



hardware：驱动代码



using：应用代码



Libraries：STM32库
